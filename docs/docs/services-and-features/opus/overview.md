# Overview

AvePoint Opus Public APIs extend AvePoint Opus functionalities with the following capabilities:

- System Integration and Content Lifecycle Management

    The APIs integrate AvePoint Opus with external data storage systems, supporting end-to-end content lifecycle management.

- In-Place Restoration and Post-Restore Retention Management

    The APIs support in-place restoration of archived data to its original locations in SharePoint Online, OneDrive, and Microsoft Teams & Groups, enabling efficient retrieval. They also support setting a retention period for restored data in archive storage, allowing users to control how long the data remains available and helping to optimize storage usage after restoration.
