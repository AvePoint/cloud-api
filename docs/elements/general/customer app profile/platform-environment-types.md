# Platform Environment Types

The following table lists the platform environment types along with their corresponding enumeration values and descriptions.

| Platform Environment Type | Enumeration Value | Description |
| ---------- | ----| --------------| 
| None | 0 | No environment specified (uninitialized or unknown). |
| AzureCommercial | 1 | Standard global Microsoft Azure cloud (public). |
| AzureChina | 2 | Microsoft Azure operated by 21Vianet in China.|
| AzureUSGovernment | 3 | Microsoft Azure Government (US GCC / GCC High). |
| Salesforce | 4 | Salesforce production (standard commercial) environment. |
| SalesforceSandbox | 5 | Salesforce sandbox (test/sandbox environment, not production). |
| AmazonCommercial | 6 | Amazon Web Services (AWS) global commercial cloud. |
| AmazonGov | 7 | AWS GovCloud (US government region). |
| AmazonChina | 8 | AWS China (Beijing/Sinnet operated regions). |
